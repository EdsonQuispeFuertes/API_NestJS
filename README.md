## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
[Project](https://github.com/EdsonQuispeFuertes/API_NestJS) Personal project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Endponint
